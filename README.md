## Author
Sophie Clemens,

## Description
`num_analyze.py` is a Python program that prompts the user for a range of numbers and analyzes each number within that range. The program categorizes numbers based on properties such as whether they are even, odd, or prime.

The results are printed to the console for each number in the specified range.

## How to Run
python num_analyze.py

python -m unittest

## Version 
The number analysis program & unit testing framework are compatible with Python 3.11.8
